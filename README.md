# Push Notification PWA (Netlify, Web Push VAPID)

## Comandos

```bash
npm install
npx netlify dev
```

## Deploy
- Fazer deploy no Netlify (publica /public e usa funções /netlify/functions)

## Notas
- Não usa Firebase.
- Usamos WebPush API + chaves VAPID.
- As notificações abrem https://www.google.com.
